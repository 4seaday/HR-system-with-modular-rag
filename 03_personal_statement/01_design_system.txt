자기소개서 평가 기능
     - 여러 기능을 수행할 수 있어야 함
     - 기능을 수행하려면 task에 적합한 프롬프트가 전달되어야 함
     - 우리가 수행할 task에 적합한 프롬프트를 미리 만들어 놓음
     - 기능 요청 -> 적합한 프롬프트를 검색할 수 있어야 함
     - 프롬프트 검색 기능 추가
     - 여러 task 설명 = 곧 프롬프트이므로 설명을 프롬프트에 전달해서 검색할 수 있도록 구성해야함

어떤 기능을 구현할 것인가?
    1) 자기소개서 질문과 답변이 얼마나 일치하는지 평가
    2) 자기소개서 내용과 회사의 핵심가치가 얼마나 부합하는지 평가
    3) 자기소개서 내용을 기반으로 해당 포지션에서 기대하는 바 생성
    4) 자기소개서 내용을 기반으로 면접 질문 생성
